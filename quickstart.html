The Collection — Quick Start

Hundreds of the greatest games ever made, ready to go. No wrong way in — grab one and press start.

How to use it

Find & play L/R switch lists · A play · B back · SELECT favorite ⭐
Switch / quit tap MENU game-switcher · double-tap = last game · hold MENU = save & quit
Fast-forward hold MENU+R (skip slow text) · MENU+L rewind
Sleep / resume tap POWER — wakes up exactly where you left off
★ THE BIG ONE — Save anywhere. Hold MENU+R2 to bookmark the exact moment, any time. MENU+L2 loads it back; MENU+←/→ picks a slot. Use it before a tough part — no progress ever lost. (Wrong load? RetroArch menu = MENU+SELECT → Undo Load State.)
