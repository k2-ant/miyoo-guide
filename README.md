# The Collection — A Field Guide to Your Handheld

A printable, single-page guide to a curated retro-handheld game library: how to use the
device, the standout games across 10 systems, and a showcase of the fan-made **ROM hacks**
that make the collection special.

- **`index.html`** — the whole site (self-contained; CSS is inline). Open it in any browser.
- **`images/`** — box art (from the collection).

### View it
Just open `index.html` locally, or host it free with **GitHub Pages** (Settings → Pages →
deploy from `main` / root). It's a static page — no build step.

### Print it
Use your browser's Print (⌘P / Ctrl-P). A print stylesheet switches to clean black-on-white
with page breaks, so it makes a nice physical booklet.

*Box art is used for identification/cataloging of a personal collection. No game ROMs are
included in this repository.*
